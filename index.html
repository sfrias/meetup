The Tarragona Developers group organizes a series of meetups to share experiences and learn from one another.

Go to the following website to check out our upcoming meetups and join the group.

This is the list of meetups and presentations done so far ordered from the most recent to the oldest.

  • All about Git meetup @ April 20, 2016.
    • Why should I be using Git by Adrian Moreno
    • Getting started with GitLab by Jordi Pujol Ahulló
    • Understanding the different Git workflows by Fernando Ripoll
  • March Meetup @ March 16, 2016.
    • Mobile games: from design to market by Jose
    • Container-Focused Operating Systems by Adrian Moreno
    • Building modern apps with scaffolding tools, by Guillermo Guerrero
  • Meetup kick-off: Introduction, Docker, and Vagrant @ February 17, 2016.

    • Managing dev environments with Vagrant by Guillermo Guerrero
    • Testing Docker with Vagrant by Jordi Massaguer Pla
